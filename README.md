## The Eco Table Project

ⅰ. 사용언어 : JAVA(JDK 17), SQL, Python

ⅱ. Database : Oracle Database 11g

ⅲ. 개발도구 : Android Studio 2022.3.1. Patch 3, Oracle SQL Developer, Colab
